# Scoop Java [![Tests](https://github.com/Unity-Billal-mesloub/Java/actions/workflows/ci.yml/badge.svg)](https://github.com/Unity-Billal-mesloub/Java/actions/workflows/ci.yml) [![Excavator](https://github.com/Unity-Billal-mesloub/Java/actions/workflows/excavator.yml/badge.svg)](https://github.com/Unity-Billal-mesloub/Java/actions/workflows/excavator.yml)

A bucket for [Scoop](https://scoop.sh), for [Oracle Java](https://www.oracle.com/java/), [OpenJDK](https://openjdk.java.net), [Eclipse Temurin](https://adoptium.net), [IBM Semeru](https://developer.ibm.com/languages/java/semeru-runtimes), [Zulu](https://www.azul.com/products/zulu-community), [Amazon Corretto](https://aws.amazon.com/corretto), [BellSoft Liberica](https://bell-sw.com/java) and [Microsoft JDK](https://www.microsoft.com/openjdk).

To make it easy to install apps from this bucket, run
    `scoop bucket add java`

For more information, read the [wiki](https://github.com/Unity-Billal-mesloub/Java/wiki).
